
INSTRUCCIONES DE COMPILACIÓN:
1. Abre una terminal o línea de comandos.

EJERCICIO 1: Listar Contenido de Directorio
----------------------------------------------
Este programa lista todos los archivos en un directorio proporcionado por el usuario, 
ordenándolos alfabéticamente.

INSTRUCCIONES DE EJECUCIÓN:
1. Compilar el ejercicio con el comando:
   javac n1exercici1/App.java n1exercici1/Directory.java n1exercici1/MyComparator.java

2. Ejecutar el ejercicio con el comando:
   java n1exercici1.App

EJERCICIO 3: Información de Directorio en Archivo
-------------------------------------------------
Este programa toma una ruta de directorio y escribe información detallada de su estructura y 
archivos a un archivo de texto, incluyendo el tipo (archivo o directorio), el nombre y la fecha de última modificación.

INSTRUCCIONES DE EJECUCIÓN:
Compilar el ejercicio con el comando:
1. javac n1exercici3/App_File.java n1exercici3/DirectoryTree_version2.java n1exercici3/FileWriterReader.java

2. Después de compilar, ejecuta el programa con este comando:
   java n1exercici3.App_File

EJERCICIO 5: Serialización y Deserialización de Objetos
--------------------------------------------------------
Este programa solicita al usuario que introduzca su nombre y edad. Luego crea un objeto que encapsula estos datos y lo serializa en un archivo. Posteriormente, recupera el objeto del archivo y muestra sus datos.


INSTRUCCIONES DE EJECUCIÓN:

1. javac n1exercici5/App.java n1exercici5/ObjectFileManager.java n1exercici5/ObjectTest.java

2. java n1exercici5.App
